## 推荐开发顺序

1. 先读 `00_项目概述_目标用户_功能范围.md`，确认 MVP 不发散。
2. 再读 `01_总体技术架构_客户端App设计.md`，确定工程骨架。
3. 第一阶段开发引用 `02_表情系统_显示器表情实现.md`。
4. 第二阶段开发引用 `03_语音系统_说话实现与打断.md`。
5. 第三阶段开发引用 `05_情绪引擎_对话系统_AI网关_API.md`。
6. 第四阶段开发引用 `04_传感器_摄像头视觉系统.md`。
7. 做页面、数据、隐私能力时引用 `06_数据结构_页面_工作模式_主动行为_安全隐私.md`。
8. 每个阶段收尾时引用 `07_开发计划_MVP任务_Prompt_测试_工程加固_风险_里程碑.md` 做验收。
